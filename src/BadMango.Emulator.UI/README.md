# BadMango.Emulator.UI

User interface components for graphical emulation display.

## Structure

- **/AvaloniaApp/** - Avalonia-based cross-platform UI (planned)

## Purpose

This project will provide graphical interfaces for:
- Text mode display (40/80 column)
- Lo-Res graphics (40x48)
- Hi-Res graphics (280x192)
- Super Hi-Res graphics (640x200, Apple IIgs)
